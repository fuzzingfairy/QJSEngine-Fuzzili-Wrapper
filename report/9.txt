Description
------------------------------------------------------------------------------
CWE-476: Out-Of-Bounds Read

- Expected behavior:
    Scoped(const Scope &scope, typename T::Data *t) validates the pointer returned by scope.alloc

- Actual behavior:
	Scoped(const Scope &scope, typename T::Data *t) dereferences a pointer without first checking it is in mapped memory resulting in an application crash via segmentation fault


Security Impact
------------------------------------------------------------------------------
Confidentiality of the process memory could be disrupted through the possibility for this leading to an arbitrary memory read


Remediation and Mitigating Factors
------------------------------------------------------------------------------
Remediation can be done by validating the pointer returned by scope.alloc in qv4scopedvalue_p.h:350.


Affected Components
------------------------------------------------------------------------------
- file:
	qtdeclarative/src/qml/jsruntime/qv4arrayobject.cpp
	qtdeclarative/src/qml/jsruntime/qv4scopedvalue_p.h
- function:
	QV4::ArrayProtype::method_reverse()


Reproduction
------------------------------------------------------------------------------
Calling of Reflect.apply with argument array length between 3900000000-3975732999:
    const v1 = [];
    const v3 = [];
    v3.length = 3900000000;
    const v5 = v1.reverse;
    const v6 = Reflect.apply(v5,v1,v3);
